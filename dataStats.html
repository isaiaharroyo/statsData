Calculating a grade

The penguin sylabus has the following information

  • Final: 35%
  • Tests: 30%
  • Quiz: 20%
  • Homework: 15%

Write a function that takes the classroom data and returns an array of objects. Each object will have two attributes: a picture and the final grade in the class.

    var classData = classroom.map(pengInfo)

    var finalgrade = function(final,test,quiz,hw,pic)
    {
        var finFinal = 0.35 * final;
        var finTest = 0.30 * test;
        var finQuiz = 0.20 * quiz;
        var finHw = 0.15 * hw;
        var finEverything = finFinal + finTest + finQuiz + finHw;
        var object = {}
            object.picture = pic;
            object.finalGrade = finEverything;
        var array = [object]
        return array;
    }

For this problem, we started by writting a function that takes the final grade, the test average, the quiz average, and the homework average. These averages came as the result of a previous function. We then created variables that were the product of the inputs and their repective percentages. We finally created a variale that was the sum of all of the products, which was the final grade in the class. We then created an empty object and added the picture and the final grade. We created an array including the object before returning the array.
